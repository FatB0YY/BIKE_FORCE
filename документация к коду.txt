struct userInfo{} хранит в себе email, password, roles, link

Struct product{} хранит в себе такие поля как: id,
name, price, rating, img, createdAt, updatedAt, categoryId, brandId, info

var productList[] хранит в себе примерные получаемые данные от сервера(готовые данные)

struct ProductMainView - показывает информацию о конкретном товаре

class CartViewModel{} - создан для таких окон как CartView и CartProductView, имеет следующие функции:
func addToCart(product: Product){} - добавляет в корзину товар и увеличивает общую цену
func removeCart(product:Product){} - удаляет продукт с корзины

struct CartView - показывает список товаров в корзины

struct CartProductView - показывает определённый товар в корзине

class screenMove - с помощью данного класса осуществляется переходы между окнами

struct MainView - главное окно, в котором показаны списки товаров, и иконка корзины

struct CardButtonView - окно корзины, оно показывает какое количество товара в корзине

struct ProductList - список товаров

struct ProductView - окно определённого товара для mainView

enterView_class - класс в котором осуществляется логика входа и регистрации, там есть такие функции как 
func passowrdValidation - проверяет пароль на требования разработчиков
func check2Password - проверяет 2 пароля на идентичность

enterView, enterView1 - окна в которых осуществляется вход и регистрация в приложение
